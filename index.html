ဆိုတာဘာလဲ ?

iERP System ဆိုတာ ရွှေနှင့် ကျောက်မျက်ရတနာဆိုင်များက စလို့ အိမ်ဆောက်ပစ္စည်းဆိုင်တွေနဲ့ စတိုးဆိုင်များပါမကျန် စနစ်တကျ လည်ပတ်စေမယ့် System တစ်ခုပဲဖြစ်ပါတယ်။ Client တွေရဲ့ လိုအပ်ချက်တွေကို Customized ပြုလုပ်ပေးထားပြီး 100% error ကင်းတဲ့ system တစ်ခုလည်း ဖြစ်ပါတယ်။

IERP SYSTEM ကို ဘာကြောင့်အသုံးပြုသင့်တာလဲ ?

Frequently Asked Questions?

iERP Software ကို အသုံးပြုဖို့ ဘာကြောင့် Memory Stick လိုတာလဲ ?

Software ကို computer မှ Installation လုပ်နိုင်ဖို့အတွက် အသုံးပြုရခြင်းပဲဖြစ်ပါတယ် ။ တကယ်လို့ Memory Stick ပျောက်ဆုံးသွားခဲ့ရင်လည်း Admin Team ကိုဆက်သွယ်လိုက်ရုံနဲ့Monery Stick အသစ်တစ်ခု ပေးသွားမှာပါ။ ဒီအတွက်တော့ Fees သက်သက်ထပ်ပေးရမှာဖြစ်ပါတယ် ။

ကိုယ်တိုင် Setup လုပ်ရမှာလား ?

iERP system ကို ဝယ်ယူလိုက်တာနဲ့ Company ဘက်က လာရောက်ပြီး Setup လုပ်ပေးမှာဖြစ်ပါတယ်။ ဒီအတွက်ကိုတော့ Setup fees ပေးစရာမလိုဘဲ Company ဘက်ကဝန်ဆောင်မှုပေးမှာဖြစ်ပါတယ် ။

iERP Software ရဲ့ Report တွေကို Online ကနေကြည့်ရင် လုံခြုံမှုရှိလား?

iERP Software က HTTPS အမျိုးအစားဖြစ်ပါတယ် ။HTTPS အမျိုးအစားဆိုတာကတော့ Data များလုံခြုံဖို့အတွက် SSL ( Secure Socket Layer ) Certificate ကို အသုံးပြုထားတာကြောင့် HTTPS ဖြစ်လာပြီး Data များကို ပိုမိုလုံခြုံစေပါတယ်။

Report တွေကရော တိကျရဲ့လား?

Accountant တွေပါ ပါဝင်စစ်ဆေးထားပြီး Accountant များကဲ့သို့ လုပ်ငန်းဆောင်တာတွေကို လုပ်ဆောင်တဲ့အတွက် report တွေက တိကျမှန်ကန်နေမှာပါ။

Customize ရောလုပ်ပေးလား ?

Clients များ၏ စိတ်တိုင်းကျ UI များပြုလုပ်ပေးပြီး လိုချင်တဲ့ Function တွေကို စိတ်တိုင်းကျပြုလုပ်ပေးပါတယ်။ သတ်မှတ်ထားတဲ့ Function ထက်ကျော်လွန်ရင်တော့ Extra Fees ထပ်မံပေးဆောင်ရမှာပဲဖြစ်ပါတယ် ။

Error တစ်စုံတစ်ရာဖြစ်လာခဲ့ရင် ဘယ်လိုဖြေရှင်းပေးမှလဲ?

iERP system ကို INNOCREATION Co.,Ltd မှ လက်ရှိလုပ်ငန်းကြီးများတွင်အသုံးပြုလျက်ရှိတဲ့ Software တွေလို စံချိန်စံညွှန်းတွေနဲ့စမ်းသပ်မှုပေါင်းများစွာလုပ်ပြီးမှ ထုတ်လုပ်ခြင်းပဲဖြစ်ပါတယ်။ Error တစ်စုံတစ်ရာပေါ်ပေါက်ခဲ့ပါကလည်း Online Update ဖြင့် INNOCREATION Co.,Ltd မှ ဖြေရှင်းပေးမှာဖြစ်ပါတယ် ။ Online Update လုပ်တယ်ဆိုတာ ဘယ်လိုမျိုးလဲ ? Online Update ထွက်လာပါက System ရဲ့ အပေါ်ဆုံး iERP logo နားမှာ A New Version Available ဆိုပြီး စာတန်းလေးပေါ်လာမှာပဲဖြစ်ပါတယ်။ အဲ့ဒီစာတန်းလေးကို click နှိပ်လိုက်တာနဲ့ iERP System တစ်ခုလုံး အလိုအလျောက် လွယ်လွယ်ကူကူ Update ပြုလုပ်ပေးသွားမှာဖြစ်ပါတယ် ။

Software Training ရောပေးလား?

ERP System Box ထဲတွင်ပါဝင်သော User Guide မှ Software Installation အချက်များကိုကြည့်ပြီး ကိုယ်တိုင်ပြုလုပ်နိုင်သည်အထါ ရေးသားထားပေးပါတယ်။ Training ယူလိုပါကလည်း INNOCREATION Co.,Ltd ကိုဆက်သွယ်ပြီး Training ရယူနိုင်ပါတယ်။
